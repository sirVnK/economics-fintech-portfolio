# Economics & Financial Technology Portfolio

Software and AI projects that connect economic data, financial analysis, forecasting, risk, and reliable application engineering.

## About

Economics & Financial Technology connects my economics background with software and AI engineering. I am building this track around transparent data pipelines, time-aware evaluation, interpretable baselines, and systems that are useful without presenting research outputs as financial advice.

## Focus Areas

- Economic data ingestion, cleaning, validation, and visualization
- Financial time-series forecasting with baseline comparisons
- Credit-risk modeling and interpretable scorecards
- Fraud and anomaly detection under class imbalance
- Portfolio optimization, backtesting, and transaction-cost awareness
- Banking workflow automation and audit-friendly data processing
- Financial APIs, caching, relational data modeling, and observability
- Responsible AI for finance, including leakage prevention and model limitations
